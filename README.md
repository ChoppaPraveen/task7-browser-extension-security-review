# Browser Extension Security Review

## Objective
To review installed browser extensions, identify potential security and privacy risks, and remove suspicious or unnecessary extensions.

## Tools Used
- Google Chrome / Mozilla Firefox
- Browser Extension Manager
- Online security research resources

## Methodology
1. Opened browser extension manager
2. Reviewed all installed extensions
3. Checked permissions for each extension
4. Verified developer authenticity and user reviews
5. Identified unused or suspicious extensions
6. Removed unnecessary extensions
7. Restarted browser to observe performance changes
8. Researched risks of malicious browser extensions

## Findings
- Some extensions requested excessive permissions
- Unused extensions increased attack surface
- Removing suspicious extensions improved browser performance

## Outcome
The browser environment became more secure and efficient after removing unnecessary extensions and reviewing permissions.

## Conclusion
Regular auditing of browser extensions is essential to protect user data, privacy, and system security.
